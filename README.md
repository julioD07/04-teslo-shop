# Teslo API

1. Clonar Proyecto
2. ```bash $ npm install ```

3. Clonar el archivo ```.env.example``` y renombrarlo a ```.env```
4. Configurar las variables de entorno

5. Levantar la base de datos
```bash
$ docker-compose up -d
 ```
